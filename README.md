# 🎒 Campus Lost & Found

A simple web-based platform that allows students to report lost and found items on campus. Users can post items, search listings by category or location, and contact the item owner or finder.

---

## 📌 Project Overview

Campus Lost & Found is a frontend web application built using **HTML, CSS, and JavaScript**. It helps students:

- Report lost items
- Post found items
- Search items by category or location
- View item details
- Contact the owner/finder
